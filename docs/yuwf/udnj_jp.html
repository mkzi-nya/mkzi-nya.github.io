蜀道の難

导览

噫吁戲ああ　危あやいかな高たかいかな
蜀道しょくどうの難かたきは青天せいてんに上のぼるよりも難かたし
蚕叢さんそうと魚鳧ぎょふと
国くにを開ひらくこと　何なんぞ茫然ぼうぜんたる
爾来じらい　四万八千歳よんまんはっせんさい
秦塞しんさいと人煙じんえんを通つうぜず
西にしのかた太白たいはくに当あたりて鳥道ちょうどうあり
以もって峨眉がびの頂いただきを横絶おうぜつすべし
地ち崩くずれ山やま摧くだけて壮士そうし死しし
然しかる後のちに天梯てんてい　石桟せきさん　方まさに鉤連こうれんす
上うえには六竜回日りくりゅうかいじつの高標こうひょうあり
下したには衝波逆折しょうはぎゃくせつの回川かいせんあり
黄鶴こうかくの飛とぶも尚なおほ過すぐるを得えず
猿猱えんのう　度わたらんと欲ほっして　攀援はんえんを愁うれう
青泥せいでい　何なんぞ盤盤ばんばんたる
百歩ひゃっぽに九折きゅうせつして巌巒がんらんを縈めぐる
参しんを扪とで井せいを歴へて　仰あおぎて脅息きょうそくし
手てを以もって膺むねを撫なでて　坐ざして長嘆ちょうたんす
君きみに問とう　西遊せいゆうして何いずれの時ときにか還かえらんと
畏途いと巉巌ざんがん　攀よづべからず
但ただだ見みる　悲鳥ひちょうの古木こぼくに号さけび
雄おすは飛とび雌めすを従したがえて　林間りんかんを遶めぐるを
又また聞きく　子規しきの夜月やげつに啼なきて空山くうざんに愁うれうるを
蜀道しょくどうの難かたきは　青天せいてんに上のぼるよりも難かたし
人ひとをして　此これを聴ききて朱顔しゅがんを凋しぼましむ
連峰れんぽう　天てんを去さること尺しゃくに盈みたず
枯松こしょう倒さかさに掛かかりて　絶壁ぜっぺきに倚よる
飛湍ひたん　瀑流ばくりゅう　争あらそいて喧豗けんかいし
崖がけを砯うち石いしを転ころじて　万壑ばんがく雷かみなりす
其その険けんなるや此かくの若ごとし
嗟ああ　爾なんじ　遠道えんどうの人ひと、胡為なんすれぞ来きたれるや
剣閣けんかくは崢嶸そうこうとして崔嵬さいかいたり
一夫いっぷ　関かんに当あたれば万夫ばんぷも開ひらく莫なし
守まもる所ところ　或あるいは親しんに匪あらずんば
化かして狼おおかみと豺やまいぬとに為ならん
朝あしたに猛虎もうこを避さけ
夕ゆうべに長蛇ちょうだを避さく
牙きばを磨みがき血ちを吮すい
人ひとを殺ころすこと麻あさの如ごとし
錦城きんじょうは楽たのしと伝つたうと雖いえども
早はやく家いえに還かえるに如しかず
蜀道しょくどうの難かたきは　青天せいてんに上のぼるよりも難かたし
身みを側そばてて西望せいぼうし　長ながく咨嗟しさす
